# cppbook

#### 介绍
《C/C++简明双链教程》电子版开发

#### 软件架构
Vue2


#### 使用方法

1.  安装nodejs，请见https://nodejs.org/zh-cn

测试是否安装成功：

```shell
node -v
```

> v16.10.0（较新版本即可）

```shell
npm -v
```

> 7.24.0（较新版本即可）

2. 安装yarn

```shell
npm install -g yarn
```

3. 安装vue-cli

```shell
yarn global add @vue/cli
```

4. 下载代码

```shell
git clone git@gitee.com:upclx/cppbook.git
```

或者

```shell
https://gitee.com/upclx/cppbook.git
```

5. 进入仓库

```shell
cd cppbook
```

6. 安装依赖

```shell
yarn install
```

7. 启动服务器

```shell
yarn serve
```

8. 打包构建

```shell
yarn build
```
